# Web3 Fullstack Application

## Deskripsi Proyek

Aplikasi ini merupakan aplikasi web full-stack sederhana yang mengintegrasikan
frontend React.js, backend Node.js/Express.js, serta interaksi dengan blockchain
Ethereum menggunakan Ethers.js dan MetaMask.

Aplikasi memungkinkan pengguna untuk:

- Menghubungkan wallet MetaMask
- Melihat saldo wallet Ethereum (testnet Sepolia)
- Melihat daftar transaksi (dummy) dari backend
- Mensimulasikan integrasi data smart contract

---

## Teknologi yang Digunakan

- Frontend: React.js (Vite)
- Backend: Node.js, Express.js
- Blockchain: Ethereum (Sepolia Testnet)
- Library Web3: Ethers.js
- Wallet: MetaMask
